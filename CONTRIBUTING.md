# Contributing

Thanks for your interest in contributing to `oss-helper-cli`.

This project aims to stay small, practical, and easy to understand. Contributions that improve clarity, usability, and maintainability are especially welcome.

## Ways to contribute

- report bugs
- suggest new CLI helpers or workflow templates
- improve wording in generated output
- improve docs and examples
- add tests
- improve packaging and release workflow

## Development setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
```

## Running the CLI locally

```bash
./.venv/bin/oss-helper --help
```

## Contribution guidelines

- keep features focused and easy to explain
- prefer clear text output over clever formatting
- avoid adding heavy dependencies unless there is a strong reason
- update the README when behavior changes

## Pull requests

Small, well-scoped pull requests are preferred.

Please include:

- what changed
- why it changed
- how you tested it
